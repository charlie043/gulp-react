# gulp-react
最低限のcompile環境

reactをお試ししたい人用

簡単のためbootstrap使用

- [React](http://facebook.github.io/react/) 

- [React-Bootstrap](http://react-bootstrap.github.io/) 

### clone
```
$ git clone https://github.com/happy-charlie-777/gulp-react.git
```
### npm install
```
$ cd gulp-react
$ npm install
```
### compile
```
$ gulp
```
